# Habit Tracker

A modern, AI-powered habit tracking application built with Next.js, featuring a clean UI, persistent data storage with MongoDB, and personalized habit suggestions via Google Gemini AI.

## 🎨 Design Credit
The UI/UX design for this project was inspired by the tutorial: [Build a Full-Stack AI-Powered Habit Tracker App](https://www.youtube.com/watch?v=PO-UZsQnkNU&list=PLbRpkalMD_hxM8lBsTiRetKkrd-QyDxEQ&index=1) (MERN Stack).

> **Note:** While the visual design is inspired by the tutorial above, the **entire technical implementation is original**. This project moves away from the traditional MERN stack in favor of a modern Next.js architecture.

## 🚀 Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Library**: [React 19](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Database**: [MongoDB](https://www.mongodb.com/) with [Mongoose](https://mongoosejs.com/)
- **Authentication**: [Clerk](https://clerk.com/)
- **AI Integration**: [Google Gemini AI](https://ai.google.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [canvas-confetti](https://www.npmjs.com/package/canvas-confetti)
- **Form Handling**: [Zod](https://zod.dev/) for validation
- **Theme**: [next-themes](https://www.npmjs.com/package/next-themes) for Dark/Light mode support

## ✨ Key Features

- **Personalized Dashboard**: A beautiful overview of your daily habits, streaks, and overall progress.
- **AI Habit Suggestions**: Tell the AI your goals and struggles, and get 3 personalized habit recommendations tailored to your lifestyle.
- **AI Weekly Reports**: Get a summary of your performance and actionable insights generated by AI based on your completion history.
- **Morning Motivation**: Start your day with an AI-generated motivational message.
- **Smart Streaks**: Automatic calculation of active and best streaks for both daily and weekly habits.
- **Interactive Progress**: Visualize your consistency with a GitHub-style activity grid and circular progress indicators.
- **Demo Mode**: Explore the full application with a pre-populated guest account—no sign-up required.
- **Multi-User Support**: Secure authentication and data isolation using Clerk.
- **Responsive Design**: Optimized for both desktop and mobile devices with a sticky mobile navigation bar.

## 🛠️ Getting Started

### Prerequisites

- Node.js 18+ 
- MongoDB instance (local or Atlas)
- Clerk account for authentication
- Google Gemini API key

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd habit-tracker
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory and add the following:
   ```env
   # MongoDB
   MONGO_URI=your_mongodb_uri

   # Clerk
   NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_publishable_key
   CLERK_SECRET_KEY=your_secret_key

   # Gemini AI
   GEMINI_API_KEY=your_gemini_api_key
   ```

4. Seed the database (optional):
   ```bash
   npm run db:seed
   ```

5. Start the development server:
   ```bash
   npm run dev
   ```

## 📁 Project Structure

- `app/`: Next.js App Router pages and layouts.
- `components/`: Reusable React components organized by feature (dashboard, nav, forms, etc.).
- `lib/`: Utilities, database models, server actions, and constant definitions.
- `public/`: Static assets.
- `proxy.ts`: Middleware for route protection and demo mode handling.
